<h1 align = "center">vue-admin-work</h1>

## 项目简介：

使用了当今最流行的技术框架： `Vue2.x + Javascript + Element UI + Webpack`



## 下载&安装

```
cd vue-admin-work
# 安装依赖
npm i
# 本地开发 启动项目
npm run serve
```
